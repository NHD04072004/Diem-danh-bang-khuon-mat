# Hệ thống điểm danh bằng khuôn mặt

### Thêm file .env vào sau đó thay đổi các thông số trong <> để kết nối với MySQL
```dotenv
DB_USER=<username_db>
DB_PASS=<password_db>
DB_HOST=<db_host>
DB_NAME=<db_name>
```
